# sonar-mobility-grid
Decentralized Vehicle-to-Vehicle (V2V) Edge AI communication system that enables real-time hazard detection, predictive alerts, and collaborative route optimization between vehicles. Inspired by sonar networks, each vehicle processes data locally and shares intelligence with nearby nodes without relying on centralized cloud systems.
